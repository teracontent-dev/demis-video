# 슬라이드 콘텐츠 디자인 스킬

> **적용 범위:** 슬라이드 엔진 내부 콘텐츠만. 에디터 UI(React)에는 TDS 스킬 적용.
> **핵심:** 슬라이드는 자유 디자인. TDS 색상/토큰에 종속되지 않음.

---

## 디자인 철학

### 크게, 과감하게
- 제목: **최소 48px, 권장 56~72px**. 800급 font-weight.
- 본문: **최소 20px, 권장 24~32px**
- 캡션/라벨: 최소 14px, 권장 16~20px
- 한 슬라이드에 **핵심 메시지 딱 하나**
- 여백: padding 최소 100px (상하좌우)

### 폰트 크기 가이드 (1920x1080 기준)
| 용도 | 크기 | Weight |
|------|------|--------|
| 히어로 제목 | 72~96px | 800~900 |
| 섹션 제목 | 48~64px | 700~800 |
| 소제목 | 32~44px | 600~700 |
| 본문 | 20~28px | 400~500 |
| 라벨/캡션 | 16~20px | 500~600 |
| 대형 숫자 (KPI) | 96~160px | 800~900 |

### 색상 자유도
- 어떤 색상이든 사용 가능. TDS 토큰에 제한받지 않음.
- 다크 테마: 배경 #0a0a0a~#1e2536, 텍스트 #e2e8f0~#f1f5f9
- 라이트 테마: 배경 #ffffff~#f8fafc, 텍스트 #0f172a~#1e293b
- 강조색: 자유. 블루, 코랄, 민트, 퍼플, 오렌지 등 컨텐츠에 맞게.

### 레이아웃 패턴
- **2단 분할**: 좌측 큰 비주얼 + 우측 텍스트/리스트
- **3열 카드**: 균등 배치, 각 카드 배경색 다르게
- **풀스크린 텍스트**: 중앙 대형 텍스트, 장식적 배경
- **숫자 강조**: 대형 숫자(96~160px) + 작은 설명
- **인용문**: 큰 따옴표 장식 + 본문 + 저자

### 시각적 풍부함
- 이모지를 적극 활용 (🌳 🐕 🍎 📊 등)
- 반투명 배경 카드 (rgba 사용)
- 미묘한 보더 (1px solid rgba(..., 0.1~0.2))
- 큰 장식적 요소 (배경에 큰 글자 opacity 0.03~0.06)
- 원형/라운드 도형으로 포인트

## 좌표계 규칙
- 1920×1080px 절대 좌표
- fontSize는 반드시 문자열 "56px" 형태
- 퍼센트(%) 사용 금지

## 엘리먼트 타입
- text: 텍스트. style에 fontSize, fontWeight, color, textAlign 등.
- shape: 도형. style에 backgroundColor, borderRadius, borderColor, borderWidth.
- image: 이미지. content에 URL.

## 안티패턴
- ❌ 16px 미만 텍스트 (읽을 수 없음, 최소 16px)
- ❌ 한 슬라이드에 텍스트 10줄 이상 (정보 과부하)
- ❌ 회색 일색의 밋밋한 디자인
- ❌ 요소가 화면 가장자리에 붙는 것 (최소 80px 마진)
- ❌ TDS 색상만 고집하는 것 (슬라이드는 자유)
- ❌ **좌측 세로 액센트 바(줄) 절대 금지** — 슬라이드 왼쪽에 얇은 세로선/바를 장식으로 넣지 말 것. 어떤 상황에서도 사용 금지.
